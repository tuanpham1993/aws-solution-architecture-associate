    - lustre
        - access
            - ec2, ecs, eks (most linux amis)
        - protocol
            - POSIX-compliant
        - fast, HPC, machine learning
        - repository
            - s3
            - on premise
        - options
            - scratch file system (ssd)
                - temporary
                - data is not replicated
                - high burst
            - persistent file system (ssd, hdd)
                - longterm
                - replicated in same az
    - netapp ontap

        - options

        - integrate Microsoft AD
    - openzfs
