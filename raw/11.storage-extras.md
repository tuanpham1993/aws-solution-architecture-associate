

- storage gateway
    - bridge between on premise and cloud
    - types
        - s3 file gateway
            - cache most recently
        - fsx file gateway
            - cache frequently access file
            - smb protocol
            - fsx window file server
        - volume gateway
            - iSCSI protocol
            - cached volumes
            - stored volumes
        - tape gateway
            - cache most recently
            - iscsi protocol

    - storage gateway hardware appliance

- aws transfer family
    - transfer file in/out s3, efs
    - FTP
    - FTPS
    - SFTP

- dataSync
    - move large data to and from
        - on premise / other cloud to aws
            - need install agent
        - aws to aws
            - keep metadata, permission
    - schedule sync
